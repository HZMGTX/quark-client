Questions

Does Quark install anything in my game folder?

No. Quark attaches to the running Java process through the Attach API. There's no mod jar, no mods folder, and nothing left behind when you close the game.

What does Quark actually do?

It's a HUD and quality-of-life client: a customizable live HUD, FullBright, Zoom and a keyboard ClickGUI. It does not include combat or movement automation — anything that couldn't be done honestly was left out rather than faked.

Which Minecraft versions are supported?

Quark targets the 1.21.x family and resolves game internals across multiple mappings, falling back gracefully so a mismatch hides a widget instead of crashing the game.

Is it really free and open source?

Yes. The full launcher and injected client are on GitHub. You can read every line, build it yourself, or contribute.
